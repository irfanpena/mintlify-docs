## Prerequisites
- Ensure that you have an active Tavus Platform account. If not, sign up [here](https://platform.tavus.io/auth/sign-up).
## Create an API Key
1. Navigate to the [Tavus platform](https://platform.tavus.io/).
2. Click the **API Key** on the sidebar navigation menu.
3. Click **Create New Key**.
4. Fill in the **API Key name** (Required) and **Whitelisted IPs** (Optional).
5. Click **Create API Key**.
6. Copy your generated API Key and store it in a safe location that you can access easily.
  <Warning>
  We cannot recover your API Key if you lose it.
  </Warning>
7. Once you have your API key, you can use the Tavus API. Follow this quickstart guide to make your first request:
    - Conversation
    - Persona
    - Replica
